About

Holybook is a non-commercial project, whose aim is to provide a well-designed interface for searching and exploring the sacred texts of the world's religions. It will be provided free of charge and without advertisements at all times. The main code base of the project is open source.

Version

The current version of holybook is 0.1. The current status represents a test version that includes only a small fraction of the writings of the Bahá'í Faith and only supports a very basic feature set such as searching and browsing the writings. The aim of this version is to provide a proof of concept that allows to test those basic features and decide on how to move on and what should be the next features.

Contribute

Currently the project is maintained by a single persion in his free time, but in the future it should be more and more community-driven. I would like to encourage everyone who thinks that this is a good project to contribute. Contribution is possible in the following ways:

  • Suggestions and Feedback: If you have any suggestions or feedback, please write a mail to feedback@holybook.ws. Please try to be constructive. At a later point, a forum will be set up to be able to discuss certain suggestions with the community.
  • Content acquisition: In order to be able to provide more sacred texts in a variety of languages, it is necessary to format these texts in a common XML format that can be consumed by the holy book service. Most of the texts will be in public domain, but in order to ensure their authenticity, it would be desirable to receive them from a authoritative entity, such as a publisher, for example. At a later stage an optional digital signature mechanism will be implemented to provide a secure mechanism of asserting the authenticity of a text. If you
    • have contacts with publishers
    • have access to holy texts
    • like to help in formatting existing content so it can be imported
    please write to content@holybook.ws
  • Design: If you are a designer and have experience in logo design or web-based user interface design and would like to contribute, please write an email to contact@holybook.ws

  • Development: The code for the project is open-source and hosted on github. In consists of multiple repositories that are collected in a common organization. The following repositories are currently hosted on github:

    • API. This project contains a nodejs server that provides an api to search and browse the text. As a storage engine it uses elastic search.
    • Webclient. This project contains the frontend for the web application. It is implemented with angularjs.
    • Rawdata. A repostiory to host all the text content in XML format.
    • Data Fetch. A project that can be used to retrieve text content and format it automatically. Currently it only supports fetching the Bahá'í Reference Library.

    As with open source projects, if you want to help with a single issue or feature, please create a pull-request on the respective repository. If you are an experienced angularjs or nodejs developer and would like to help with the development on a more regular basis, please write an E-Mail to contact@holybook.ws. Also, if you encounter a problem please create an issue on the respective repository.

Roadmap

The features planned for the next version (0.2) include

  • Ranking of search results, that is incrementally optimized over time depending on the actions of users.
  • Multi-language support.
  • Extend the content to cover more books in different languages.
  • When searching, filter by author or religion.
